# HappyMac 🖥️

> 一个吸附在书桌背景板上的赛博小生命体。  
> 它不是工具，它只是在陪着你。

---

## 项目愿景

HappyMac 是一枚桌面像素交互小宠物，形态为冰箱贴。它极度克制、省电，平时几乎沉默，但在你需要的瞬间，能给出精准而温柔的回应。

它有自己的作息，有自己的情绪，会在你深夜还没睡的时候困着眼撑着陪你，也会在你发现它睡着的瞬间上演一出惊慌失措的假装没睡。

没有摄像头，没有麦克风，只有热量和微波的反射——它用这两样东西感受你的存在。

## 硬件架构

```
┌─────────────────────────────────────┐
│  OLED 1.3"  SH1106  128×64         │  ← 像素脸输出
│  ESP32-C3 SuperMini                 │  ← 核心大脑
│                                     │
│  能量分级感知系统                    │
│  ├─ 第一级：SR602 红外传感器         │  ← 哨兵，平时深度休眠
│  ├─ 第二级：LD2410C 毫米波雷达       │  ← 微动/能量/呼吸感知
│  └─ 第三级：LD2450 毫米波雷达        │  ← X/Y 双目标追踪
│                                     │
│  电源管理                            │
│  ├─ TP4056 双输入充电（USB-C + 太阳能）│
│  ├─ SS14 防反流二极管                │
│  └─ AO3401 高侧开关（雷达电源门控）  │
└─────────────────────────────────────┘
```

### 为什么是两个雷达

LD2410C 和 LD2450 分工不同，互相不可替代：

| 传感器 | 擅长 | 在本项目中的角色 |
|--------|------|-----------------|
| LD2410C | 微动能量、呼吸、静止检测 | 判断用户状态（专注/发呆/睡着） |
| LD2450 | X/Y 坐标、多目标追踪、速度 | 判断用户位置和大幅度动作 |

## 交互设计

### 伪 3D 视差跟随

HappyMac 的脸由三个层次构成：双眼、鼻子、嘴巴。当用户位置偏移时，三个层次以不同幅度位移，形成视差效果：

```
用户向左移动时：
  鼻子  移动 ±6px   （最前层，移动最大）
  眼睛  移动 ±4px   （中层）
  嘴巴  移动 ±2px   （最后层，移动最小）
```

在 2D 屏幕上实现"它在转头盯着你"的视觉感受，无需 3D 硬件。

### 状态机

```
用户出现         → 慢慢从睡眠中醒来
用户看它         → 惊喜/脸红/眼睛睁大
用户发呆         → 跟着一起发呆，眼神涣散
用户长时间不理   → 打哈欠 → 揉眼 → 亮度渐暗 → 睡眠呼吸动画
深夜用户还在     → 困意动画，但撑着陪你
用户离位         → 目送 → 等待 → 睡着
被发现在睡觉     → 惊醒 → 假装没睡
```

### 睡眠状态的设计原则

全黑 = 坏掉了。HappyMac 睡着时：
- 亮度降至 20%，不是 0
- 慢速呼吸闪烁，表示有生命迹象
- 有人靠近时先有微小反应，再慢慢睁眼

## TinyML 训练路径（盲眼剑客计划）

最终产品绝对不包含摄像头，但训练阶段需要视觉作为 Ground Truth。

### 三步走

**第一步：师徒结对（数据采集）**

引入电脑摄像头作为"上帝视角"。MediaPipe 实时识别头部姿态和注视方向，自动打标签；同步记录同一时刻的雷达波形和红外特征。标注过程完全自动，用户正常使用电脑即可完成被动采集。

**第二步：炼丹提取（模型训练）**

将多模态数据输入神经网络，训练两个独立模型：

```
模型 A：位置动作分类
  输入：LD2450 的 X/Y + 帧间变化量
  输出：坐下/站立/前倾/后靠/左偏/右偏
  驱动：HappyMac 脸的朝向

模型 B：用户状态分类
  输入：LD2410C 能量值 + LD2450 速度辅助
  输出：专注/活跃/发呆/睡着/离位
  驱动：HappyMac 表情 + 屏幕亮度
```

两个模型时间尺度不同，分开训练，通过优先级仲裁层协同工作。

**第三步：过河拆桥（边缘部署）**

将模型量化部署进 ESP32-C3，彻底拔掉摄像头。设备仅凭热量和微波反射，凭直觉感受主人的目光。

## 设计哲学

> 它不应该每天都需要你。  
> 但当你抬头的那一刻，它刚好也在看你——  
> 这就够了。

隐私优先：无摄像头，无麦克风，无联网，无云端。感知只发生在本地，数据从不离开桌面。

省电优先：三级唤醒结构，绝大多数时间处于深度休眠。太阳能辅助充电，真正的低打扰存在。

克制优先：去掉所有工具属性（无时钟、无天气、无通知推送），只保留陪伴本身。
